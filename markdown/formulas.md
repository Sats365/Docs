В тексте можно использовать формулы.

| Формула                          | Код                               |
| -------------------------------- | --------------------------------- |
| $L = \frac{1}{2} \rho v^2 S C_L$ | `$L = \frac{1}{2} rho v^2 S C_L$` |

Подробнее о синтаксисе читайте [тут](https://github.com/micromark/micromark-extension-math#syntax).
